# 🌙 Todo Luna

Современное полнофункциональное приложение для управления задачами, созданное на **Nuxt 3** и **IndexedDB** для автономного хранения данных.

![Nuxt](https://img.shields.io/badge/Nuxt-3.x-00DC82?style=flat-square&logo=nuxt.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?style=flat-square&logo=typescript)
![IndexedDB](https://img.shields.io/badge/IndexedDB-Dexie-3498DB?style=flat-square)

---

## ✨ Возможности

- 📝 **Rich Text Редактор** — Редактор на базе Tiptap с форматированием, ссылками, блоками кода и многим другим
- 🗂️ **Группы задач** — Организация задач в настраиваемые группы с эмодзи
- 📊 **Статистика** — Отслеживание выполненных, активных и всех задач по группам
- 💾 **Автономная работа** — Все данные хранятся локально в IndexedDB (через Dexie)
- 🎨 **Современный UI** — Чистый адаптивный дизайн с плавными анимациями
- ⌨️ **Горячие клавиши** — Быстрые действия для опытных пользователей (например, `Ctrl+L` для создания группы)
- 🔍 **Умная навигация** — Контекстно-зависимая маршрутизация
- 📱 **Адаптивность** — Бесперебойная работа на десктопе, планшете и телефоне

## 📦 Установка

### Требования

- Node.js 18+ 
- pnpm (`npm install -g pnpm`)

### Настройка

```bash
# Перейдите в директорию src
cd src

# Установите зависимости
pnpm install
```

## 🚀 Запуск приложения

### Режим разработки

```bash
cd src

# Запуск сервера разработки на http://localhost:3000
pnpm dev

# Доступ из сети (для тестирования на других устройствах)
pnpm dev-host

# С туннелем для внешнего доступа
pnpm dev-host-tunnel
```

### Продакшен сборка

```bash
cd src

# Сборка для продакшена
pnpm build

# Предварительный просмотр продакшен сборки
pnpm preview
```

## 🐳 Docker развёртывание

Проект включает многоэтапный Dockerfile для оптимизированных продакшен сборок:

```bash
# Сборка Docker образа
docker build -f .docker/Dockerfile -t todo-luna .

# Запуск контейнера
docker run -p 3000:3500 todo-luna
```

Контейнер работает с **PM2** для управления процессами с 2 рабочими экземплярами.

## 💾 Хранение данных

Todo Luna использует **IndexedDB** в качестве основного решения для хранения данных:

- ✅ **Без бэкенда** — Все данные хранятся локально в браузере
- ✅ **Автономная работа** — Работает без подключения к интернету
- ✅ **Постоянные данные** — Данные сохраняются после перезапуска браузера
- ✅ **Большая вместимость** — Может хранить значительно больше данных, чем localStorage
- ✅ **Быстрые запросы** — IndexedDB обеспечивает эффективное извлечение данных

База данных управляется через **Dexie.js** — легковесную обёртку для IndexedDB, которая упрощает операции и предоставляет чистый API.

## 🎯 Основные страницы

| Маршрут | Описание |
|---------|----------|
| `/` | Главная страница со всеми задачами |
| `/group/:slug` | Задачи, отфильтрованные по группе |
| `/group/empty-group` | Задачи без группы |
| `/group/:slug/:toDoSlug` | Редактор задачи |
| `/empty-group/:toDoSlug` | Редактор для задач без группы |
